01
// Contact Us
ติดต่อและแจ้งความต้องการ
ทัก DM มาที่ Instagram หรือ TikTok @black1gothic พร้อมแจ้งข้อมูลเบื้องต้นดังนี้
  • ไอเดียหรือธีมที่ต้องการ เช่น Cybersigilism, Gothic, หรือผสมผสาน
  • ตำแหน่งที่ต้องการเพ้นบนเสื้อ เช่น หน้า / หลัง / แขน
  • ขนาดของลาย (คร่าวๆ)
  • รูปเสื้อที่จะนำมาเพ้น ถ่ายภาพส่งมาได้เลย
02
// Receive Quote
รับ Quote และยืนยันงาน
ทางร้านจะประเมินราคาและระยะเวลาให้ภายใน 24-48 ชั่วโมง โดยพิจารณาจากความซับซ้อนของลายและขนาดงาน เมื่อลูกค้ายืนยันรับราคาแล้ว จึงดำเนินการขั้นตอนถัดไป
03
// Deposit Payment
ชำระเงินมัดจำ
เพื่อยืนยันการจอง ลูกค้าชำระ มัดจำ 50% ของราคาทั้งหมด ก่อนเริ่มงาน
  • โอนผ่านธนาคาร (แจ้งบัญชีทาง DM)
  • PromptPay
📌 งานจะเริ่มดำเนินการหลังได้รับหลักฐานการโอนเท่านั้น
04
// Send Your Shirt
ส่งเสื้อมาที่ร้าน
ลูกค้าจัดส่งเสื้อมาที่ร้านด้วยตัวเอง โดย ค่าขนส่งขาไปเป็นความรับผิดชอบของลูกค้า
📌 แนะนำให้แพ็คเสื้อให้มิดชิด และถ่ายภาพก่อนส่งเพื่อเป็นหลักฐาน
05
// Approve Design
ออกแบบและยืนยันลาย
ทางร้านออกแบบลายเป็น ไฟล์ดิจิทัล และส่งให้ลูกค้าดูก่อนลงมือเพ้นจริง
  • ✅ ยืนยันลาย → ทางร้านเริ่มเพ้นทันที
  • 🔄 ปรับแก้ได้ไม่จำกัดครั้ง ภายใต้ concept เดิม ไม่มีค่าใช้จ่ายเพิ่ม
📌 หากต้องการเปลี่ยน concept ใหม่ทั้งหมด อาจมีค่าออกแบบเพิ่มเติมตามความซับซ้อน
06
// Paint & Update
เพ้นและอัปเดตความคืบหน้า
ทางร้านดำเนินการเพ้นตามลายที่ยืนยันแล้ว และจะ อัปเดตภาพระหว่างทำ ให้ลูกค้าติดตามความคืบหน้าได้ตลอด
⏱ ระยะเวลาโดยประมาณ 5-10 วันทำการ ขึ้นอยู่กับความซับซ้อนของงาน
07
// Final Payment & Ship
ชำระเงินส่วนที่เหลือและจัดส่ง
เมื่องานเสร็จสมบูรณ์ ทางร้านจะส่งภาพผลงานให้ลูกค้าดูก่อน จากนั้น
  • ลูกค้าชำระ ยอดที่เหลือ 50%
  • ทางร้านจัดส่งเสื้อกลับ โดย ค่าขนส่งขากลับเป็นความรับผิดชอบของลูกค้า
  • แจ้งเลขพัสดุให้ทาง DM
08
// Receive & Review
รับเสื้อและ Review
ลูกค้ารับเสื้อและตรวจสอบความเรียบร้อย หากมีปัญหาจากการขนส่งหรือความผิดพลาดจากทางร้าน สามารถแจ้งได้ภายใน 48 ชั่วโมง หลังได้รับสินค้า
