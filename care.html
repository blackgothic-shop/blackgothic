// First Wash Rule
รอ 72 ชั่วโมงหลังได้รับเสื้อก่อนซักครั้งแรก

แม้ลายจะผ่านกระบวนการ heat-set มาแล้ว แต่สีอะคริลิคบนผ้ายังต้องการเวลาเพื่อยึดติดกับเส้นใยผ้าให้แน่นสมบูรณ์ การซักเร็วเกินไปอาจทำให้สีซึมออกหรือลายเริ่มแตกตั้งแต่การซักครั้งแรก

หลีกเลี่ยงฝนช่วงแรก ไม่โดนน้ำโดยตรง เก็บในที่แห้ง ยิ่งรอนาน ยิ่งดี
// คู่มือดูแลรายหัวข้อ
ทุกขั้นตอน
อย่างละเอียด
01
ซักมือ
// Hand Wash — แนะนำที่สุด
  • ·
    กลับด้านในออกก่อนซักทุกครั้ง เสมอ
  • ·
    ใช้น้ำเย็นเท่านั้นน้ำอุ่นหรือร้อนทำให้สีซีดและลายแตก
  • ·
    ใช้ผงซักฟอกอ่อน สำหรับผ้าบางหรือ Delicate
  • ·
    กดเบา ๆ ด้วยมือ — ห้ามขยี้หรือถูบริเวณลายแรงเสียดทานโดยตรงคือสาเหตุหลักที่ทำให้ลายหลุด
  • ·
    ห้ามแช่น้ำนาน — สีซึมออกจากลายได้
  • ·
    บีบน้ำออกเบา ๆ ห้ามบิดเป็นเกลียว
02
ซักเครื่อง
// Machine Wash — ถ้าจำเป็น
  • ·
    กลับด้านในออกก่อนใส่เครื่องทุกครั้ง
  • ·
    ใส่ถุงซักผ้า (Laundry Bag)ป้องกันลายเสียดสีกับผ้าชิ้นอื่น
  • ·
    เลือก Gentle / Delicate Cycle เท่านั้น
  • ·
    น้ำเย็น (Cold Water) เสมอ
  • ·
    ใช้ผงซักฟอกอ่อน ห้ามใช้น้ำยาฟอกขาวทุกชนิด
  • ·
    ห้ามซักรวมกับผ้าที่มีซิปหรือตะขอขูดและทำลายลายได้
03
การอบผ้า
// Tumble Dry — หลีกเลี่ยงที่สุด
  • ·
    ห้ามใช้เครื่องอบผ้า — หลักการสำคัญที่สุดความร้อนทำให้ผ้าหดและสีขยายตัวต่างอัตรา = ลายแตกทันที
  • ·
    ถ้าจำเป็นต้องอบ → กลับด้านในก่อนเสมอ
  • ·
    ใช้ความร้อนต่ำสุด (Low Heat) หรือ Air-Only
  • ·
    ตรวจสอบลายทุก 10 นาที อย่าปล่อยทิ้งไว้
  • ·
    นำออกทันทีที่แห้ง อย่าปล่อยให้ความร้อนค้าง
04
การตาก
// Air Drying — วิธีที่ดีที่สุด
  • ·
    ตากในที่ร่ม อากาศถ่ายเทดี เสมอ
  • ·
    วางราบบนผ้าสะอาดหรือตะแกรงวิธีที่ดีที่สุด รักษาทรงและป้องกันผ้ายืด
  • ·
    ถ้าแขวน → แขวนที่ชายเสื้อด้านล่าง ไม่ใช่หัวไหล่
  • ·
    ห้ามตากโดนแสงแดดจัดโดยตรงUV ทำให้สีซีดจางเร็ว โดยเฉพาะสีเข้มและสีขาว
  • ·
    รอแห้ง 100% สนิทก่อนพับเก็บความชื้นค้างทำให้เกิดเชื้อราและกลิ่น
05
การรีด
// Ironing — ระวังเป็นพิเศษ
  • ·
    ห้ามรีดทับลายโดยตรงเด็ดขาดลายละลาย ติดเตารีด หรือเปลี่ยนสีถาวร
  • ·
    กลับด้านในออกก่อนรีดทุกครั้ง
  • ·
    รองผ้าบาง (มัสลิน / กระดาษรองรีด) ระหว่างเตารีดกับเสื้อ
  • ·
    ใช้ความร้อนต่ำ (Synthetic / Low)
  • ·
    ห้ามใช้ไอน้ำ (Steam) บริเวณลายโดยตรงไอน้ำซึมเข้าลายและทำให้สียึดติดเสื่อมเร็ว
  • ·
    กด 3–5 วินาทีต่อส่วน ไม่ลากไปมาบนลาย
06
การเก็บรักษา
// Storage — ระยะยาว
  • ·
    เก็บในที่เย็น แห้ง อากาศถ่ายเทได้ ห่างแสงแดด
  • ·
    ห้ามเก็บในที่ชื้นเชื้อราทำลายลายและผ้าอย่างถาวร
  • ·
    พับ: หันลายออกด้านนอก ไม่พับทับบริเวณลายโดยตรง
  • ·
    แขวน: ใช้ไม้แขวนที่กว้าง ไม่บีบบ่า รักษาทรงเสื้อ
  • ·
    ไม่ยัดอัดแน่นในที่แคบแรงกดสะสมทำให้ลายแตกร้าวในระยะยาว
  • ·
    เก็บระยะยาว → ห่อด้วยผ้ามัสลินหรือกระดาษกรดฟรี (Acid-Free)
// Spot Cleaning
// ทำความสะอาดเฉพาะจุด
เปื้อนนิดหน่อย
ไม่ต้องซักทั้งตัว
// ขั้นตอน Spot Clean
  • 01
    ระบุจุดที่เปื้อน — อยู่บนลายหรือผ้าเปล่า? ถ้าบนลายต้องระวังมากขึ้น
  • 02
    ใช้ผ้าสะอาดชุบน้ำเย็นบิดหมาด — ไม่เปียกมากเกินไป
  • 03
    กดซับเบา ๆ บนจุดที่เปื้อน — ห้ามถู ห้ามขยี้โดยเด็ดขาด
  • 04
    ถ้าจำเป็น เติมผงซักฟอกอ่อน 1–2 หยด ลงบนผ้าเช็ด แล้วกดซับซ้ำ
  • 05
    ใช้ผ้าชุบน้ำสะอาดกดซับอีกครั้งเพื่อล้างผงซักฟอกออก
  • 06
    ตากในที่ร่มให้แห้งสนิท 100% ก่อนพับเก็บหรือใส่
// ทำไมต้อง Spot Clean?

การซักทุกครั้งทำให้ลายเสื่อมสภาพทีละนิด แม้จะซักถูกวิธีก็ตาม น้ำ แรงเสียดทาน และผงซักฟอก ล้วนมีผลต่อความทนทานของสีในระยะยาว

Spot Cleaning ช่วยดูแลความสะอาดเฉพาะจุดโดยไม่ต้องนำเสื้อผ่านกระบวนการซักทั้งหมด ยืดอายุการใช้งานของลายได้อย่างมีนัยสำคัญ

⚠ สำคัญ: ถ้าเปื้อนบริเวณลายโดยตรง ให้กดซับเบาที่สุดเท่าที่เป็นไปได้ และทดสอบในจุดเล็ก ๆ ก่อนเสมอ
จำไว้เสมอ: เสื้อเพ้นลายมือคืองานศิลปะ ไม่ใช่เสื้อทั่วไป ลายทุกเส้นใช้เวลาและทักษะในการสร้าง — ยิ่งดูแลดีเท่าไหร่ ลายก็ยิ่งอยู่ทนและสวยงามได้นานขึ้นเท่านั้น
